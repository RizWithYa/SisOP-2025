# <u> Jawaban beserta Penjelasan </u> :


## 1. Pengertian Bilangan Biner dan Bilangan Heksa

### a. Bilangan Biner
Bilangan biner merupakan bilangan yang berbasis dua. Bilangan ini digunakan sebagai sinyal elektronik untuk pemrosesan data karena hanya memiliki 2 simbol berbeda, yakni **0** dan **1**.

### b. Bilangan Heksa
Bilangan heksa banyak digunakan dalam komputer. Sistem bilangan ini terdiri dari 16 simbol, yaitu 10 angka (0–9) dan 6 huruf (A–F) yang nilainya sebagai berikut:
- **A = 10**
- **B = 11**
- **C = 12**
- **D = 13**
- **E = 14**
- **F = 15**

---

## 2. Konversi Desimal ke Biner

### a. Contoh: Konversi 1234₁₀ ke Biner
Untuk mengkonversi bilangan desimal **1234** ke biner, lakukan pembagian berulang dengan 2, dan catat sisa bagi setiap langkahnya.

Pembagian:
1234 ÷ 2 = 617 sisa 0
617 ÷ 2 = 308 sisa 1
308 ÷ 2 = 154 sisa 0
154 ÷ 2 = 77 sisa 0
77 ÷ 2 = 38 sisa 1
38 ÷ 2 = 19 sisa 0
19 ÷ 2 = 9 sisa 1
9 ÷ 2 = 4 sisa 1
4 ÷ 2 = 2 sisa 0
2 ÷ 2 = 1 sisa 0
1 ÷ 2 = 0 sisa 1

Kemudian, ambil angka sisa dari bawah (hasil akhir pembagian) untuk mendapatkan bilangan biner:
  
**Hasil:** `10011010010`

---

## 3. Konversi Biner ke Desimal dengan Pendekatan LSB

### b. Contoh Penggunaan LSB (Least Significant Bit)
Misalkan diberikan kode biner seperti:
- **01010101**

Untuk mengkonversi bilangan biner ke desimal, kita menghitung dari bit paling kanan (LSB). Contoh konversi untuk bilangan **10101010** adalah sebagai berikut:

Dari bit terakhir ke bit pertama:
- **Bit ke-0:** 1 × 2⁰ = 1  
- **Bit ke-1:** 0 × 2¹ = 0  
- **Bit ke-2:** 1 × 2² = 4  
- **Bit ke-3:** 0 × 2³ = 0  
- **Bit ke-4:** 1 × 2⁴ = 16  
- **Bit ke-5:** 0 × 2⁵ = 0  
- **Bit ke-6:** 1 × 2⁶ = 64  
- **Bit ke-7:** 0 × 2⁷ = 0  

Jumlahkan semua hasil perkalian:
  
1 + 0 + 4 + 0 + 16 + 0 + 64 + 0 = **85**

---

## 4. Konversi Biner ke Oktal

### a. Contoh: Konversi `101011111001₂` ke Oktal
Ikuti langkah-langkah berikut untuk mengkonversi bilangan biner ke oktal:

1. **Pengelompokan Digit**  
   Kelompokkan digit biner menjadi grup yang masing-masing berisi 3 digit, dimulai dari kanan.  
   Jika jumlah digit tidak kelipatan 3, tambahkan 0 di sisi kiri.  
   Dalam contoh ini, jumlah digit adalah 12 (kelipatan 3), sehingga pembagian langsung:
   101 011 111 001

3. **Konversi Setiap Grup ke Nilai Oktal**  
- **Grup `101₂`:**  
  1 × 2² + 0 × 2¹ + 1 × 2⁰ = 4 + 0 + 1 = **5**
- **Grup `011₂`:**  
  0 × 2² + 1 × 2¹ + 1 × 2⁰ = 0 + 2 + 1 = **3**
- **Grup `111₂`:**  
  1 × 2² + 1 × 2¹ + 1 × 2⁰ = 4 + 2 + 1 = **7**
- **Grup `001₂`:**  
  0 × 2² + 0 × 2¹ + 1 × 2⁰ = 0 + 0 + 1 = **1**

3. **Gabungkan Digit Oktal**  
Setelah konversi, gabungkan nilai-nilai oktal tersebut:

Sehingga, diperoleh:

**`101011111001₂ = 5371₈`**
5 3 7 1

## 6. Konversi Desimal ke Heksadesimal

**Soal:**  
Konversikan bilangan desimal **1780₁₀** ke heksadesimal.

**Langkah-langkah pembagian dengan 16:**

1. **Pembagian pertama:**  
1780 ÷ 16 = 111 dengan sisa **4**

2. **Pembagian kedua:**  
111 ÷ 16 = 6 dengan sisa **15**  
- Catatan: Nilai sisa 15 dalam heksadesimal adalah **F**.

3. **Pembagian ketiga:**  
6 ÷ 16 = 0 dengan sisa **6**

4. **Pengambilan sisa dari bawah ke atas:**  
- Sisa terakhir (paling atas): 6  
- Sisa berikutnya: F  
- Sisa pertama: 4  

Sehingga bilangan heksadesimal yang diperoleh adalah **6F4**.  
Jika diperlukan 4 digit, tambahkan 0 di depan sehingga menjadi **06F4**.
